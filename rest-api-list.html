Cloud-Barista User REST API (v0.4.0-cafemocha)

 

[클라우드 인프라 연동 정보 통합 관리 (CB-Spider)]

  • 클라우드 인프라 연결을 위한 정보 등록 및 관리
  • API 규격/샘플: https://cloud-barista.github.io/rest-api/v0.4.0/spider/ccim/
  • Maintainer: @powerkimhub

[멀티 클라우드 인프라 서비스 통합 관리 (CB-Tumblebug)]

  • 네임스페이스 생성 및 관리, MCIR 등록/생성 및 관리, MCIS 생성/제어 및 관리
  • API 규격/샘플: https://cloud-barista.github.io/rest-api/v0.4.0/tumblebug/
  • Maintainer: @seokho-son @jihoon-seo

[멀티 클라우드 애플리케이션 실행환경 통합 관리 (CB-Ladybug)]

  • 쿠버네티스 클러스터 생성/삭제/정보 조회, 노드 추가/삭제/정보 조회
  • API 규격/샘플: https://cloud-barista.github.io/rest-api/v0.4.0/mcks/
  • Maintainer: @sykim-etri @itnpeople

[멀티 클라우드 인프라 서비스 통합 모니터링 (CB-Dragonfly)]

  • MCIS 및 VM 모니터링(온디멘드 모니터링 조회, MCIS 모니터링 조회, 알람 이벤트 핸들러 관리 및 알람 관리)
  • API 규격/샘플: https://cloud-barista.github.io/rest-api/v0.4.0/dragonfly/
  • Maintainer: @hyokyungk

  • API 규격/샘플:
    • - Image : [MCIR Image API]
    - Spec : [MCIR Spec API]
    - VNet : [MCIR VNet API]
    - SecurityGroup : [MCIR SecurityGroup API]
    - AccessKey : [MCIR AccessKey API]
  • Maintainer: @jihoon-seo @seokho-son

  • API 규격/샘플:
    • - 생성 및 기본 제어 : [MCIR Provisioning and Lifecycle API]
    - 정책 기반 자동 관리 : [MCIR Auto Management API]
  • Maintainer: @seokho-son @jihoon-seo
